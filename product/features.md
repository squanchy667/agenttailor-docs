# Features

## Core Features

- **Visual Agent Builder** — Drag-and-drop interface for composing agent system prompts, personality, and behavioral rules
- **Tool Configuration** — Define tools/functions with schema validation for each platform
- **Cross-Platform Export** — Generate configs for ChatGPT (Custom GPTs, Assistants API) and Claude (Projects, API)
- **Template Library** — Pre-built agent templates for common use cases
- **Version Control** — Track configuration changes with diff view
- **Live Preview** — Simulated conversations to test agent behavior

## Planned Features

- Import/reverse-engineer existing agent configs
- Team collaboration with shared agent libraries
- Deployment automation (push configs directly to platforms via API)
- Analytics dashboard for agent performance comparison
- Plugin system for custom export targets
