# Features

## Core Features

- **Smart Context Assembly** — Automatically analyzes tasks, retrieves relevant info from documents and web, assembles optimal context packages
- **Document Ingestion** — Upload PDF, DOCX, MD, TXT, and code files with semantic chunking and embedding
- **Cross-Platform Delivery** — Browser extension (ChatGPT + Claude), MCP Server (Claude Desktop), Custom GPT (GPT Store), REST API
- **Context Intelligence** — Cross-encoder reranking, gap detection, hierarchical compression, source synthesis
- **Web Search Integration** — Fills knowledge gaps via Tavily/Brave search, processes and merges results
- **Transparency** — Every context piece shows its source, relevance score, and why it was included
- **Quality Scoring** — Scores on coverage, diversity, relevance, and compression ratio
- **Citation Tracking** — Traces every chunk to its source document, page, heading, or web URL

## Planned Features

- Team collaboration with shared project libraries
- Analytics dashboard for context quality trends
- Plugin system for custom context sources (Notion, Confluence, GitHub)
- Streaming context assembly for real-time preview
- Self-hosted option for enterprise privacy requirements
